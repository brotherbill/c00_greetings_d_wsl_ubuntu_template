# WSL Ubuntu Setup Guide (Beginner)

This guide is for beginners. It explains every step in simple language and provides extra commentary to help you understand what you are doing and why.

## What is WSL?
WSL stands for Windows Subsystem for Linux. It lets you run Ubuntu (a popular Linux system) on your Windows computer, without needing a separate computer or a virtual machine.

## Why use Ubuntu with Windows?
Ubuntu is used by many programmers. With WSL, you can learn and use Ubuntu while still using your familiar Windows environment.

## Step-by-step Setup
1. Open PowerShell as Administrator. This gives you permission to install software.
2. Run `wsl --install`. This installs WSL. If you see any errors, follow the instructions to fix them.
3. Restart your computer. This makes sure the installation is complete.
4. Install Ubuntu by running `wsl --install -d Ubuntu` in PowerShell. This downloads Ubuntu.
5. When Ubuntu starts, create a username and password. Remember these—they are for Ubuntu, not Windows.

## What next?
Follow the instructions in this guide. If you get stuck, ask for help or search online. Every step is important, so take your time!
