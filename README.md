# D Project Template: Documentation Sequence

To get started and make the most of this template, follow these documentation files in order:

1. **WSL_Ubuntu_Setup_Guide.md**
   - Step-by-step instructions for setting up **WSL** and **Ubuntu** on **Windows**, including all prerequisites for D development.

2. **create_new_d_project.md**
   - How to create a new D project from this template, with detailed Bash script usage and project organization tips.

3. **wsl_ubuntu_vs_windows.md**
   - Key differences, tips, and best practices for working between WSL Ubuntu and Windows environments.

Refer to each guide in sequence for a smooth setup and project creation experience.
